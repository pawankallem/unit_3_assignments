OBJECT

  • Defination:an object is a standalone entity, with properties and type
  • Example:Compare it with a cup, for example. A cup is an object, with properties. A cup has a color, a design, weight, a material it is made of, etc.
  • JavaScript objects can have properties, which define their characteristics.
  • Inside Object,It containes Key-Value Pairs
  • Some Inbuilt Function: Object.keys(),Object.values(),Object.freeze();

ARRAY

  • Defination: The JavaScript Array class is a global object that is used in the construction of arrays; which are high-level, list-like objects.
  • An array is a special variable, which can hold more than one value
  • Exaple:var cars = [ "Saab", "Volvo", "BMW" ];
  • Arrays stores Data in format of 1D,2D and 3D
  • Inbuilt Functions: Array.pop(),Array.push(),Array.shift()

FUNCTION

  • Defination: A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).
  • Variables declared within a JavaScript function, become LOCAL to the function. Local variables can only be accessed from within the function.
  • Syntax: function myFunction() { alert("Hello World!"); }
  • All The Inbuilt Functions are Designed using Functions and We can Invoke them Using NameofFunction().

STRING

  • String is A Premitive Data Type
  • Defination: The String object is used to represent and manipulate a sequence of characters.
  • Syntax: let string="This is Creation Of string";

NUMBER

  • Number is a primitive wrapper object used to represent and manipulate numbers like 37 or -9.25.
  • Number()
    Creates a new Number value.
  • To Convert String to Number :
    Number(input) Or +(input);
  • We can give number input without quotes.

NULL

  • Defination: The value null represents the intentional absence of any object value. It is one of JavaScript's primitive values and is treated as falsy for boolean operations.
  • Null is a Empty Object.
  • You assign this value to the variable or object to wipe out all the data in it.
  • It’s just a special value which represents “nothing”, “empty” or “value unknown”.

UNDEFINED

  • The special value undefined also stands apart. It makes a type of its own, just like null.
  • The meaning of undefined is “value is not assigned”.
  • If a variable is declared, but not assigned, then its value is undefined.
